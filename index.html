Comics Annotation Tool (CAT)

Prototype Version 1.0

This is a web-based annotation tool for marking up comics. See the instructions below for an overview of how to use the CAT. After reading the instructions, click the Start button below to begin. Thanks for giving it a try!

How to Use the CAT:

Press Start Annotation to begin.

You will be given a number of comic pages, which are displayed one at a time on the left of the page.

For each page, outline the distinct image sections by dragging and dropping rectangles directly on the image. Press the "Start Task" to begin outlining sections.


If you make a mistake or want to redo any rectangles, use the Clear Last Rectangle and Clear All Rectangles buttons.

Once the sections are all outlined to satisfaction, click the "End Task" button.


After ending the task, a series of forms are generated on the right side of the page. Each form corresponds with each outlined section, and the number of each section matches to the number on the top left of the rectangle outlines.

Pressing the "Indicate Characters" button allows for clicking in the specified image section. Place a blue dot on each key character.

Finally, fill out each of the sections on the generated forms.


Once the entire page is annotated, click the Next Page button. You can use the Previous Page button at any time - the inputs will be saved and you can check any labels and descriptions. Click the Reset button anytime to restart the tasks on that page.
